# CliNum1 - Unit Converter

Simple unit converter web app built with Express and EJS.

This project for https://roadmap.sh/projects/unit-converter

## What It Does

- Convert weight
- Convert length
- Convert temperature

## Run

```bash
npm install
npm start
```

Open: http://localhost:3000

For development mode:

```bash
npm run dev
```

## Main Routes

- GET /
- POST /weight
- POST /length
- POST /temperature
